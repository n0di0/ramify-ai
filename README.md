▪ Problem: 
    There is a lack in the use of educational resources.
▪ Target User:
    Students attending Winston-Salem State University.
▪ Value:
     Fill the gaps in educational resources with ONE app.  
▪ How data/AI is used:
  A multifactored study tool united within a singluar mobile application:
    1. Documents the Student’s Major, Courses, and Progress 
    2. Understands WSSU’s Actual Courses and Requirements 
    3. Break Down Assignments Based on the Syllabus 
    4. Generates Study Materials Based on the Student’s Learning Style 
    5. Connects Students to Campus Resources  
    6. Integrates With the Smart Advisor Features
